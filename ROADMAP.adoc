= Package Publishers Roadmap
:toc:
:toclevels: 3

== Overview

Package Publishers is a monorepo containing publishing definitions for various package managers and platform stores, making hyperpolymath software accessible across platforms.

== Components

=== homebrew-tap (60% complete)

*Status:* Active maintenance +
*Purpose:* Homebrew formulas for hyperpolymath packages

*Current features:*
- Formula definitions for core tools
- Automated dependency tracking
- Version pinning

*Planned improvements:*
- Automated formula generation from releases
- CI testing on macOS
- Bottle pre-compilation

=== macports-ports (50% complete)

*Status:* Active maintenance +
*Purpose:* MacPorts portfiles for hyperpolymath packages

*Current features:*
- Portfile definitions
- Dependency management
- Multi-variant support

*Planned improvements:*
- Automated portfile updates
- CI validation
- Better variant documentation

=== winget-pkgs (40% complete)

*Status:* Active development +
*Purpose:* Windows Package Manager manifests

*Current features:*
- Basic manifest structure
- Version tracking

*Planned improvements:*
- Complete manifest coverage
- Automated manifest generation
- CI validation with winget
