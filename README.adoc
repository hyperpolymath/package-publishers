= Package Publishers
// SPDX-License-Identifier: PMPL-1.0-or-later

Publishing definitions for distributing Hyperpolymath tools across package managers and platform stores. A federated packaging hub covering macOS, Linux, and Windows ecosystems.

== Components

=== `homebrew-tap/` — Federated Packaging Hub

Contains definitions for multiple package managers:

[cols="2,1,3",options="header"]
|===
| Manager | Directory | Packages

| Homebrew (macOS/Linux)
| `Formula/`
| affinescript, bunsenite, casket-ssg, czech-file-knife, echidnabot, kith, llm-unify, vext, zigzag-ssg

| AUR (Arch Linux)
| `aur/`
| affinescript, bunsenite, bunsenite-bin, casket-ssg, czech-file-knife, echidnabot, kith, llm-unify, vext, zigzag-ssg

| Scoop (Windows)
| `bucket/`
| bunsenite, czech-file-knife, echidnabot, llm-unify, vext

| Flatpak
| `flathub/`
| Bunsenite, CzechFileKnife

| Nix
| `nix/`
| bunsenite, llm-unify, vext (+ flake.nix)

| Snap (Linux)
| `snap/`
| bunsenite, llm-unify, vext

| WinGet (Windows)
| `winget/`
| Bunsenite, CzechFileKnife, Echidnabot
|===

=== `macports-ports/` — MacPorts Portfiles

Full MacPorts ports tree structure with 60+ categories mirroring the upstream ports layout.

=== `winget-pkgs/` — Windows Package Manager

Extended WinGet package definitions with manifests, schemas, DevOps pipelines, and tooling.
